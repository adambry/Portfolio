Anlise do mercado de venda de carros

Realizao do projeto: Andr Dambry

O objetivo desse projeto analisar um site de venda de carros tanto novos quanto usados

No caso, o projeto vai ser baseado nos dados obtidos no site Kavak, onde ser feito um Web Scrapping para obter informaes relevantes

O projeto vai ser dividido em 3 sees, sendo elas:

  1. Aquisio e limpeza dos dados
  2. Anlise dos dados
  3. Concluso

O resultado final esperado de encontrar, a partir dos dados obtidos no site, qual seria o carro mais rentvel para a revenda, ou seja, qual seria o carro com o maior valor e tambm com o maior nmero de anncios

Antes de realizar as anlises, sero consideradas algumas premissas com o intuito de simplificar a anlise com os dados adquiridos


    

  1. Os dados adquiridos na KAVAK refletem o estado geral do mercado de venda e revenda de veculos
    2.

2.1 Contagem de anncios por carro
Foi feita a contagem de cada anncio de carro e criada uma coluna com a porcentagem acumulada  partir nmero da contagem, assim foi pego os carros que representam 25% da base de dados

df_count_car_all = df.groupby(['Carro', 'Marca']).count()

Sem considerar a categoria, fica bvio que ambos os carros Compass e Renegade da Jeep so aqueles que tm a maior demanda, porm isso tambm pode ser explicado pela pouca variedade de carros da marca, mas isso ainda assim no tira o fato que so carros com muita demanda.

Assim, para um investimento menor, o Ford Ka e Hyundai HB20 so uma opo de incio de investimento para revenda, alm dos carros da VW (Up, Gol e Fox) que apresentam o maior nmero de anncios do site.


Outro fator de se levar em conta,  em relao se o carros esto fora de linha ou no, pois, caso esteja fora de linha, isso torna menos atrativo para o comprador, pois carros fora de linha tm menos peas disponveis no mercado, o que pode encarecer a troca delas.


Com isso, observando o artigo do Auto Esporte,  visto que o Ford Ka  um carro que j saiu de linha, logo no seria recomendado para a revenda, ento o carro popular que seria mais adequado para a revenda, seria o Hyundai HB20
